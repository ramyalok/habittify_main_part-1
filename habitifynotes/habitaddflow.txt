createdAt stores the date when the habit was created (it never changes).
history stores every date on which the habit was completed.
streak is recalculated from the history object each time the user marks a habit
.
The updated habit is saved in MongoDB, returned to the frontend, 
stored in Redux, and all components 
that use the Redux state (HabitCard, HabitStats, Calendar, 
and Analytics Dashboard) automatically re-render with the new data.
