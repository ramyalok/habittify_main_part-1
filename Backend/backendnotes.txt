Flow:

Login
↓
Backend creates token
↓
Frontend stores token
↓
User clicks Logout
↓
Frontend removes token
↓
Redux clears currentUser
↓
Navigate("/")

Example Navbar:

const handleLogout = () => {

localStorage.removeItem("token");

dispatch(logout());

navigate("/");

};

Your Redux reducer:

logout:(state)=>{

state.currentUser=null;

state.isAuthenticated=false;

}
